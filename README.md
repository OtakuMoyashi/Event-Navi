# kinshi-navi

kinshi-navi is a web application created for the 'Kinshi Festival', the cultural festival of Nagano High School in Nagano Prefecture.

kinshi-naviは長野県長野高校の文化祭『金鵄祭』のために作成されたWebアプリです。現在開発中です。
